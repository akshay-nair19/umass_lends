# UMass Lends

Welcome to UMass Lends! Here is a secure, reliable and helpful platform for you to find anything you are looking to borrow temporarily. 
We know how expensive college can be and we want to make it easier for you to save that extra buck!
Find or request what you are looking for and enjoy!
